HTML

  • HTML handles the structure and content of a webpage
  • An HTML file must always start with the tag "!DOCTYPE html", followed by "html lang="en""
  • The head element contains information about the webpage.
  • The head is used primarily by the browser to render the page properly
  • The body element represents the visible content shown to the user.

css

  • CSS defines the colors, aesthetics, visual layout, and organization
  • CSS can be added "in-line', internally, or externaly
  • If the CSS is done internally, it must be inside the Head element and within the Style tag
  • If the CSS is added externally, the style.css file must be referenced inside the Head element
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Github is an effective version control system
  • Version control keeps track of all changes to the work and allows us to go back in time
  • Github also provides a way to share work and collaborate with others

JavaScript

  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive
  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function must be called to execute
